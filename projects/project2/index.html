Bio-Sequencing

Describes each method in the Bio-sequencing app.
Method Description
Count nucleotides Calculates the total amount of each A, T, C, and G nucleotide in a given DNA sequence.
GC / AT content Calculates the percentage G & C or A & T nucleotides in a DNA given sequence.
Transcribe Converts a given DNA sequence into an RNA sequence.
Reverse-transcribe Converts a given RNA sequence into an DNA sequence.
cDNA Converts a given RNA sequence into a comlementary DNA strand.
Translate Converts a given RNA sequence into a protein.
Count amino acids Calculates the total amount of each amino acid in a protein sequence. Possible amino acids in a sequence are:
R, H, K, D, E, S, T, N, Q, C, U, G, P, A, V, I, L, M, F, Y, and W.
Protein mass Calculates the mass of a protein sequence using the following formula:
total mass of protein in Da - (18.0153 * (amount of proteins - 1)).
